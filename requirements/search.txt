Title: Browse and search for available products

As a potential customer
I want to search and explore products on ecommerce-playground.lambdatest.io
So that I can find a card that suits my needs

Acceptance Criteria:
User can navigate to the product section of the site
User can view a list of available products
User can use filters to refine results
User can click on a product to view details
User can see clear card comparison or highlights without logging in
User receives relevant results based on selected filters or search criteria
